# Product Overview

## Project Purpose
Personal developer portfolio for Sarath P, a Full Stack Software Engineer. Showcases professional experience, projects, skills, publications, and contact information to potential employers and collaborators.

## Value Proposition
A fast, accessible, and visually polished static portfolio site built with modern web technologies. Demonstrates the owner's technical expertise through the quality of the site itself — accessibility compliance, performance optimisation, and clean TypeScript code.

## Key Features
- **Hero / Home page** — animated experience counter, specialisation tags, expandable work history timeline
- **Projects page** — showcase of personal and professional projects
- **Skills page** — categorised technical skill listing
- **About page** — personal background and bio
- **Publications page** — IEEE and academic publication listing
- **Contact page** — contact form or links for reaching out
- **Responsive Navbar** — site-wide navigation with active-link highlighting

## Target Users
- Recruiters and hiring managers evaluating Sarath's candidacy
- Engineers and collaborators exploring his open-source or professional work
- Conference/publication reviewers looking up his research background

## Use Cases
- First impression landing page for job applications
- Portfolio link shared on LinkedIn / GitHub / resume
- Demonstration of frontend engineering skills (React, TypeScript, accessibility, performance)
